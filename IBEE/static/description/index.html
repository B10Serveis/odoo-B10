Usage

To use this module, you need to:

    Aquet model permet contabilitzar l'Impost de les Begues Ensucrades Envasades

    Un cop instal·lat el mòdul obrirem l'apartat INVENTARI dins d'aquet a DADES PRINCIPALS el qual surtira un desplegable amb el nom de PRODUCTES, dins ell creariem el nostre producte, el qual ens servirà per a seleccionar el percentatge del IBEE segons si és el 0.10% o el 0.15%, el qual calculara l'impost segons els litres de cada article.

    Tot seguit ens desplaçarem a l'apartat de la FACTURACIÓ en la qual crearem una factura, afegirem l'article creat anteriorment, el qual afegin-hi la quantitat de l'article desitjada ens generarà automàticament el valor de l'IBEE.

    Un cop validada la factura, ens desplaçarem al apartat superior, buscarem l'apartat INFORMES, dins ell apareixera un desplegable el qual surtira el IBEE del Model 520. En aquet recuadre fixarem les dates per obtindre l'impost de l'IBEE.

    Un cop validades les dates ens generara l'informe automàticament calculant l'impost de sucre per litre(IBEE).
